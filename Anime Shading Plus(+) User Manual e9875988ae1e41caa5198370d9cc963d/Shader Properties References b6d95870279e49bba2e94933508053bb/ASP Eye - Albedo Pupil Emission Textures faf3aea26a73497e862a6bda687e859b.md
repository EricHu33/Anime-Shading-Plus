# ASP/Eye - Albedo/Pupil/Emission Textures

## 參數說明

---

| Property Name | Description |
| --- | --- |
| Albedo | albedo map |
| Enable Parallax Effect | Enable parallax effect. you can adjust Parallax Height to create a concave effect after enable this function. |
| Pupil Mask | Mask of the pupil area, which affects the area and strength of parallax effect. It use R channel to sample the mask value. |
| Parallax Height | Parallax effect height |
| Enable Emission | Whether to enable emission |
| Emission | Emission color/map |
